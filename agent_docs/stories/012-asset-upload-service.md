# Story: 012 - Asset Upload Service

## 📖 Description
Implement a helper to upload local images generated by the AI agent to the Strapi Media Library during seeding.

## ✅ User Acceptance Criteria (UAC)
- [ ] The seeder can successfully upload a file from a local path to Strapi.
- [ ] Uploaded files appear in the Strapi Media Library with correct metadata.

## 🛠️ Technical Acceptance Criteria (TAC)
- [ ] Use `strapi.plugins.upload.services.upload.upload` service.
- [ ] Handle potential errors (e.g., file not found).

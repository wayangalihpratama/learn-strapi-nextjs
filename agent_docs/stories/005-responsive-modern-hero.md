# [Story-005] Responsive Modern Hero Component

## Description
As a user, I want a visually stunning and responsive hero section so that I can easily discover travel destinations on any device.

## User Acceptance Criteria (UAC)
- [ ] Hero displays a high-resolution immersive image with a smooth fade-in.
- [ ] Search bar is centered and uses glassmorphism effects.
- [ ] On mobile, the search bar stacks vertically and text scales down appropriately.
- [ ] Headline uses the new premium serif typography.

## Technical Acceptance Criteria (TAC)
- [ ] Use Next.js `next/image` with `priority` and `object-fit: cover`.
- [ ] Search bar component is refactored for a minimal, clean aesthetic.
- [ ] Breakpoints (sm, md, lg) are correctly applied to the hero layout.
- [ ] Performance: LCP (Largest Contentful Paint) is under 2.5s.

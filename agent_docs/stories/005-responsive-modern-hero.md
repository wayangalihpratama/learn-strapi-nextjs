# [Story-005] Responsive Modern Hero Component

## Description
As a user, I want a visually stunning and responsive hero section so that I can easily discover travel destinations on any device.

## User Acceptance Criteria (UAC)
- [x] Hero displays a high-resolution immersive image with a smooth fade-in.
- [x] Search bar is centered and uses glassmorphism effects.
- [x] On mobile, the search bar stacks vertically and text scales down appropriately.
- [x] Headline uses the new premium serif typography.

## Technical Acceptance Criteria (TAC)
- [x] Use Next.js `next/image` with `priority` and `object-fit: cover`.
- [x] Search bar component is refactored for a minimal, clean aesthetic.
- [x] Breakpoints (sm, md, lg) are correctly applied to the hero layout.
- [x] Performance: LCP (Largest Contentful Paint) is under 2.5s.
